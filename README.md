# SafeScroll
AI-powered personalized YouTube Shorts filtering with custom viewing modes.


> **Your Feed. Your Rules. Powered by AI.**

SafeScroll is an AI-powered content filtering system designed to personalize the YouTube Shorts experience through custom viewing modes.

## ✨ Features

- Personalized Viewing Modes
- Study Mode
- Child Mode
- Islamic Mode
- Custom Mode Builder
- Analytics Dashboard
- Device Management
- Security & Privacy Pages
- Responsive User Interface

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
