Clear rules for using SafeScroll.

These terms describe the intended conditions for the SafeScroll product before public deployment.

ServiceAccountsChild ModeLimitationsTermination

1. Service description

SafeScroll provides mode-based feed filtering, synchronization and personalization tools for supported browser environments.

2. Accounts and security

Users are responsible for maintaining accurate account information and protecting their credentials.

3. Child Mode

Child Mode can protect mode switching with a parent PIN, but it cannot guarantee protection if a user manually disables or removes the extension from browser settings.

4. AI and filtering limitations

AI classification can produce false positives and false negatives. SafeScroll is designed to reduce unwanted content, not guarantee perfect content safety.

5. Account termination

Users may request account deletion. Access may also be restricted for abuse, unauthorized use or attempts to compromise the service.
